## Project Description

This project was created as part of our academic courses related to business administration and computer technology. It demonstrates how students collaborate using GitHub in a real-world development environment.

## Features

* Home Page
* Group Members Section
* About Page
* Skills List

## Tools Used

* GitHub for version control
* HTML for website structure
* CSS for design and styling

## Workflow

* Each member created their own branch
* Changes were committed with proper messages
* Pull requests were submitted for review
* The project leader reviewed and merged contributions

## Purpose of the Activity

This activity serves as a practical application of digital transformation within an administrative and academic context. It aims to bridge the gap between manual paperwork and streamlined digital workflows by utilizing GitHub for collaborative version control. Through this project, students develop essential technical skills—including branching, merging, and pull requests—that are critical for managing modern business systems and real-world software development.
